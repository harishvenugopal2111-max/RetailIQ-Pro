# RetailIQ Pro

AI Powered Business Intelligence Dashboard built using Streamlit.

## Features
- Dashboard
- Analytics
- Forecast
- AI Insights
- Customer Segmentation
- Report Generator
- Login Authentication

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-Learn
- SQLite

## Installation

pip install -r requirements.txt

streamlit run app.py
